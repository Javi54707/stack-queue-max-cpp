Detailed Description

T.D.A. MaxStack.

Una instancia del tipo de dato abstracto MaxStack nos permite almacenar una cola que contiene datos de tipo element para poder simular una pila.

El TDA MaxStack proporciona además distintas herramientas para la manipulación de dicha cola, con métodos push,pop,TamPIla,top y EstaVacia.

Para poder usar el TDA MaxStack hay que incluir el fichero #include <maxstack.h>
