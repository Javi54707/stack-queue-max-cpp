Detailed Description

TDA MaxQueue Simulación de una cola a través de una lista que almacena además el máximo de los datos de la cola, guardando en cada posición un par de enteros, siendo uno el valor y otro el máximo de la cola, en un struct de tipo element.

Para poder usar el TDA MaxQueue se debe incluir: #include "maxqueue.h"
